このプログラムは3つに分かれています。ProchainePosition, TransformeEnGraphique, \
すぐ上にある１１の行。
 
 
 ProchainePosition は英語で通訳すれば NextPosition となります。この関数は
{ListeCorp_, MV_, XPV_, YPV_, ZPV_, CXVV_, CYVV_, CZVV_, dt_, 
   ListePoint_, t_} をインプットとして、{ListeCorp2, MV, XPV2, YPV2, ZPV2, 
   CXVV2, CYVV2, CZVV2, dt, ListePoint2, t2} をアウトプットとしています。
 簡単に言うと全部の体の位置、重さと速度をインプットしてdt後の全部の体の位置、重さと速度をアウトプットしています。
 この関数の中では何も変えないでください。
 変えていいものは：If[
   Mod[t2, dt*150] == 
    0] の150と0です。150 を小さくすれば赤いボールのパスにある白い点が一つずつ近づきます。0 \
をー1にすればパスは見えなくなります。
 			  Collision = True のTrueです。Falseにすれば衝突があってもシムレイションは止まりません。
 
 
 TransformeEnGraphiqueはProchainePositionのアウトプットから見える絵を作ります。
 この関数の中では何も変えないでください。
 変えていいものは：{Red, 
   Sphere[{XPV, YPV, ZPV}, 
    10]} の中のREDをちがう色にするか10を替えるか。これは動く赤いボールの色と大きさを替えます。
 			 {-1000, 1000}, {-1000, 
  1000}, {-1000, 
   1000} \
の数字のところだけです。これは視覚化の大きさを替えますｘ、ｙ、ｚとして作られています。いきなり天体が消えたりした
		   ら視覚化が大きすぎる可能性が高いです。
 
 最後の11行は一番あなたが使うところです。
 帰れるのは二つあります：
 一つ目はProchaineEtapeのインプットです。
 その形は {ListeCorp_, MV_, XPV_, YPV_, ZPV_, CXVV_, CYVV_, CZVV_, dt_, 
   ListePoint_, t_} これです。
 まずListeCorp以外のを見てみましょう。
 tはいつも0としといてください、ListePointもいつも ｛｝ \
としておいてください、dtは一人一人必要なように変えてください100000がちょうどよかったそうなので最初はそれで試してみてください。
 残りのインプットは全ての赤い天体の最初の位置、重さ、速度を表してるのでやりたいことによってお好きに変えてください。
 MVは赤いボールの重さです。変えてもいいですがあまり意味がないです。
 残りのインプットは変数としていますので跡でもっと情報を出します。
 ListeCorpsは７成分ベクトルの配列です。そのかたちは：
 {｛XP, YP, ZP, M, C, XS, YS, ZS｝, ｛XP, YP, ZP, M, C, XS, YS, ZS｝, 
  ｛XP, YP, ZP, M, C, XS, YS, ZS｝ ...}
ベクトルの最初の三つの成分はいち、そのあとは重さ、そのあとは色、残りは速度です。さいしょは
{{0, 0, 0, 100, Green, -0.000004, 0, 0}, {-500, -500, -500, 100, Blue,
     0, 0, 0}, {700, 700, 700, 100, 
    Purple, -0.000001, -0.000001, -0.000001}} とかいてあるので。
 もっと天体を作りたかったらここに  , {XP, YP, ZP, M, C, XS, YS, 
    ZS}  を入れる ------------------------------------------------------ \
- ^
   少なくしたかったら一つ取り除く。
 
 ふたつめはManipulateの変数です。それはこういう感じで出てきます：{vx, 0}, -0.00001, 0.00001
変えてもいいのは全部です。0 を替えたら最初出てくるVXのバリューが変わります。二つ目はスライダーの一番下、三つめはスライダーの一番上。
 注意：{n, 0}, 0, 5000, 
 1 は5000で毛を替えてください。nは最初の位置から何歩したかを数えるのであげれば上げるほど長くシムレーションを見れます。です\
が上げれば上げるほど時間がかかるので
制度をなくす代わりにdtをあげてもいいです。
